# CORA-Emotional-Support-Robot

CORA is a bear-shaped emotional support robot designed to assist individuals through emotional regulation and AI-assisted companionship.

------------

#Project Goals:
- Build an emotionally interactive robotics platform
- Combine embedded systems with AI
- Create a comforting robotic companion
- Explore human-robot interaction

------------

#Current Features:
- OLED animated eyes
- ESP32-based control system
- Sensor Integration

---------------

#Hardware

- ESP32
- OLED Display
- Sensors
- Servo Motors
- Speaker modules

-------------

#Software

- Arduino IDE
- C++
- Embedded Systems Programming

-----------
